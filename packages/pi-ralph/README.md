# pi-ralph

Pi extension that adopts a technique for running AI coding agents in a loop.

## Install

```bash
pi install npm:@andrii-k6a/pi-ralph
```

Or try without installing:

```bash
pi -e npm:@andrii-k6a/pi-ralph
```

Or from this repo:

```bash
pi -e ./packages/pi-ralph
```

## Uninstall

```bash
pi remove npm:@andrii-k6a/pi-ralph
```
